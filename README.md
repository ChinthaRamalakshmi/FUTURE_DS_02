# 🚀 FUTURE_DS_02 - Customer Retention & Churn Analysis

## 📊 Project Title

Customer Retention & Churn Analysis

## 🎯 Objective

To analyze customer churn patterns, identify factors influencing customer retention, and provide recommendations to reduce customer loss.

## 🛠 Tools Used

* Python
* Pandas
* Matplotlib
* VS Code

## 📁 Dataset Description

The dataset contains customer information such as demographics, contract type, internet services, payment methods, monthly charges, and churn status.

## 📊 Key Findings

### 👥 Customer Overview

* Total Customers: 7043
* Retained Customers: 5174
* Churned Customers: 1869
* Churn Rate: 26.54%

### 📄 Contract-wise Churn Analysis

* Month-to-month customers showed the highest churn (1655 customers).
* One-year contracts had 166 churned customers.
* Two-year contracts had only 48 churned customers.

### 🌐 Internet Service-wise Churn Analysis

* Fiber Optic customers had the highest churn (1297 customers).
* DSL customers had 459 churned customers.
* Customers without internet service had only 113 churned customers.

### 💳 Payment Method-wise Churn Analysis

* Electronic Check users showed the highest churn (1071 customers).
* Mailed Check users had 308 churned customers.
* Bank Transfer (Automatic) users had 258 churned customers.
* Credit Card (Automatic) users had 232 churned customers.

## 📈 Visualizations

* Customer Churn Distribution (Pie Chart)
* Contract-wise Churn Analysis
* Internet Service-wise Churn Analysis
* Payment Method-wise Churn Analysis

## 💡 Recommendations

1. Encourage customers to switch to long-term contracts.
2. Improve service quality for Fiber Optic customers.
3. Promote automatic payment methods.
4. Offer loyalty rewards and retention programs.

## 🏁 Conclusion

The analysis identified key factors influencing customer churn. Long-term contracts, better customer service, and improved retention strategies can help reduce churn and increase customer satisfaction.
